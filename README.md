# ERP Prensas PWA Universal V6.6.3

Cambios:

- Producción compacta: muestra todas las prensas y la fila TOTAL en la misma pantalla sin depender de scroll.
- Botón pequeño de compartir dentro de la tarjeta de Producción. Comparte solo la tabla actual y respeta el filtro A1/A3/Todas.
- Registro de uso a Firebase retrasado después de la carga para no competir con las APIs.
- Compatible con iPhone Safari y Android Chrome como PWA.

Subir a GitHub Pages en la raíz del repositorio: `index.html`, `manifest.json`, `sw.js`, `.nojekyll` e `icons/`.
Abrir con `?v=663` para forzar refresco.
