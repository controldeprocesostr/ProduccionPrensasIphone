# ERP Prensas PWA Universal V6.6.1

Paquete para GitHub Pages compatible con iPhone y Android Chrome.

## Subir a GitHub
Sube estos archivos en la raíz del repositorio:

- index.html
- manifest.json
- sw.js
- .nojekyll
- icons/

## Android
Abre el enlace en Chrome y usa Instalar app o Agregar a pantalla principal.
Si no aparece de inmediato, abre el enlace con ?v=661 y espera unos segundos.

## iPhone
Abre el enlace en Safari y usa Compartir > Agregar a pantalla de inicio.
